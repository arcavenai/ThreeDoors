### Pre-PR Submission Checklist

Before pushing your branch or creating a PR, complete every item:

- [ ] **Rebase onto latest main**: Run `git fetch upstream main && git rebase upstream/main` — resolve any conflicts before pushing
- [ ] **Run gofumpt**: Run `gofumpt -l .` and verify **no output** — if files are listed, run `gofumpt -w .` to fix
- [ ] **Run golangci-lint**: Run `golangci-lint run ./...` and verify **0 issues**
- [ ] **Run all tests**: Run `go test ./... -count=1` and verify **0 failures**
- [ ] **Check for dead code**: Run `go vet ./...` and review for unused functions, variables, or imports
- [ ] **Single clean commit preferred**: Squash or amend fix-ups before pushing
